# Voucher

Voucher aims to be a reference [Quorra](https://github.com/Quorra-Auth/server) keychain implementation.

## What is a keychain?

Keychains are applications created for handling your [LNURL-auth](https://lightninglogin.live/learn) login keys. Unlike wallets, they can't hold any currency and can't pay. They are only made for logging in.

## Who are keychains for?

For anyone who wants to use LNURL-auth but doesn't want a Lightning wallet!

## Supported features

- [x] LNURL-auth compatibility
- [x] Secure key storage (libsecret)
- [ ] Mnemonic backup and restore
- [ ] Key management
- [ ] Extended details about OIDC logins
